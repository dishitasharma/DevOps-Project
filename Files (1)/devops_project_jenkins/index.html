CLI
Command Line Usage
# List all templates
jenkins-pipeline-lib list

# Get details on a template
jenkins-pipeline-lib info python-ci

# Generate files into your project
jenkins-pipeline-lib generate python-ci --output ./my-project

# Get a recommendation
jenkins-pipeline-lib recommend -d "Python Django REST API with Docker" -t python -t docker

# Search templates
jenkins-pipeline-lib search docker

# List categories
jenkins-pipeline-lib categories
REST API
Endpoints
